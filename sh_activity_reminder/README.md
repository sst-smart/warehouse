About
============
This module allows to set an activity alarm that reminds of to-do various activities. You can set and send gentle reminders to perform activities so no need to remember daily tasks. You can send activity reminders at a particular date and time by email as well as popup. Activity reminders can be sent by schedule actions.

Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list.
5) search module name and hit install button.
